# Console Color Style
Package for the purpose of providing colored outputs on the console
## Quick Start

### Install
```bash
# using npm
$ npm install console-color-style
```
```bash
# using yarn
$ yarn add console-color-style
```
## Supported Colors
- **blue**;
- **red**;
- **black**;
- **green**; 
- **yellow**; 
- **magneta**; 
- **cyan** ;
- **white**; 
- **default**.
